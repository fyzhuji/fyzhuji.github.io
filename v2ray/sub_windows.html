  • v2rayN for Windows 教程
一.下载客户端 v2rayN for Windows
https://help.fengye.la/res/v2rayN.zip

解压zip文件,解压完点击打开

双击打开v2rayN。如果出现防火墙阻拦，请点击允许访问网络

登录客户中心，点击我的服务 点击套餐名称进入套餐详情页面, 点击【节点订阅】，复制订阅地址


点击按钮后会自动复制订阅地址,弹出成功提示


回到v2rayN软件，点击【订阅】，【订阅设置】，将复制的订阅地址添加至地址中，点击【确定】。


再点击【订阅】，【更新订阅】，在信息栏中看到“更新订阅结束”即可。


点击【更新】，【检查更新 v2rayCore】，在【信息】中显示‘解析v2rayCore成功‘，并且弹出是否下载，选择【是】。


等待数秒，当【信息】中显示“更新v2rayCore成功”。


注：如果提示错误无法下载，可以手动下载,然后解压到同目录。


Win X64

https://github.com/v2ray/v2ray-core/releases/download/v4.20.0/v2ray-windows-64.zip

Win X32


https://github.com/v2ray/v2ray-core/releases/download/v4.20.0/v2ray-windows-32.zip


解压之后在同目录，如下图



电脑右下角，右键软件，勾选【启用Http代理】和【Http代理模式】下的【开启PAC，并自动配置PAC】，即可使用。



注：PAC模式，大陆IP不走代理；全局模式，所有数据均走代理。



注：如果提示 PAC启动失败,请使用管理员模式打开,您需要安装 Microsoft .NET Framework 4.6或者更高的版本

下载地址 ：https://dotnet.microsoft.com/download/thank-you/net46

下载完成之后安装，安装完成之后请重启电脑，然后重新打开即可
