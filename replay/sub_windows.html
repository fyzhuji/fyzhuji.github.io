  • ShadowsocksR for Windows 教程
一.下载客户端 ShadowsocksR for Windows
https://help.fengye.la/res/ShadowsocksR-win.zip

解压zip文件,解压完出现ShadowsocksR-4.7.0-win

Windows XP 用户请使用 ShadowsocksR-dotnet2.0。Win7 系统以上用户请使用 ShadowsocksR-dotnet4.0版本


如果系统没有安装过 Microsoft .NET Framework。会提示报错，下载 Microsoft .NET Framework 安装就可以了

最新版本的4.0.6版本需要安装 .NET Framework 4.6.2 下载地址

https://www.microsoft.com/zh-CN/download/details.aspx?id=53344


双击打开ShadowsocksR。如果出现防火墙阻拦，请点击允许访问网络

软件启动之后会在右下角出现一个白色的小飞机。


在软件上鼠标右键点击，出现 服务器订阅 ->SSR订阅设置,点击进入订阅设置界面


登录客户中心，点击我的服务 点击套餐名称进入套餐详情页面。在「自动配置」功能区找到桌面平台 - 「节点订阅」。


点击按钮后会自动复制订阅地址,弹出成功提示


请删除软件默认自动的订阅地址

将刚刚复制成功的连接地址粘贴进去，然后点击确定按钮 确定。


在右下角软件上鼠标右键点击，出现 服务器订阅 ->更新SSR服务器订阅。点击进行更新


成功之后右下角会提示更新成功


如果没有提示更新成功或者提示更新失败！

请选择 服务器订阅 -> 更新SSR服务器订阅(不通过代理) 模式 。点击进行更新


在右下角软件上鼠标右键点击 在出现服务器 哪里可以看到刚刚出现的分组 FYZHUJI 点击进入选择一个节点进行确认


在右下角软件上鼠标右键点击 在系统代理模式 哪里 把模式设置为 pac 模式


如果无法访问一些网站,您可以在系统代理模式哪里 把模式设置为 全局模式。然后使用IE浏览器进行测试
