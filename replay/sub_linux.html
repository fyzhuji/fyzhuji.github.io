  • ShadowsocksR for Ubuntu 教程
一.下载客户端 ShadowsocksR for Linux
https://help.fengye.la/res/electron-ssr-0.2.3-x86_64.AppImage

然后在Applications 内搜索Terminal ,然后打开

找到下载文件目录。cd Downloads 执行 chmod a+x electron-ssr-0.2.3-x86_64.AppImage 赋予权限


安装 libsodium

执行 sudo apt-get install libsodium-dev,输入密码，然后 Y 执行安装

找到下载目录，鼠标右键 Run 打开执行


点击服务器 订阅管理


登录客户中心，点击我的服务 点击套餐名称进入套餐详情页面。在「自动配置」功能区找到桌面平台 - 「节点订阅」。

点击按钮后会自动复制订阅地址,弹出成功提示


然后在软件订阅管理 点击 添加按钮。填入复制的订阅地址，然后回车确认


更新完成之后，会自动显示订阅地址和组名


点击服务器 选择组名，然后选择一个节点进行确认


然后使用浏览器打开进行测试吧


如果还无法打开,请在帮助内 点击 查看日记 来 查看出现的错误日记
